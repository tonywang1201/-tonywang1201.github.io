spigot服务器安装EssentialsX插件

EssentialsX插件资源汇总：https://www.mcbbs.net/thread-1369748-1-1.html

EssentialsX 是一个基于 Spigot 服务端的基础插件，为从大到小的服务器提供核心功能。这些功能包括： 玩家可以自由设置家 服务器传送或给玩家提供物品套组，可以设置跨世界或单独世界。 玩家与玩家间的私有消息，传送，发送传送请求 玩家自定义昵称 很多的管理员工具包括踢出服务器，临时禁止登陆服务器、禁言与监禁 内建经济系统，包括木牌商店、付费执行命令和完全的 Vault 支持

此外，EssentialsX 的选择模块提供了更多综合的功能如聊天、世界保护、GeoIP查找还有更多

EssentialsX 是原先 Essentials 项目 的分支，Essentials 已于2014年停止更新。EssentialsX 相比 Essentials 和 Spigot 上的 Essentials 分支添加了一些新特性。

1.在EssentialsX官网下载插件

EssentialsX基础系列插件包括多个.jar文件：

  • EssentialsX
  • EssentialsX Chat
  • EssentialsX Spawn
  • EssentialsX Discord
  • EssentialsX AntiBuild
  • ……

下载后把文件放入服务器目录下的plugins文件夹

2. 启动插件

启动插件后，文件夹里会出现Essentials文件夹，里面包含了控制功能的.yml文件

3. 使用插件

直接进游戏连接服务器，就可以使用插件提供的额外命令。由于插件提供的功能很多，后续可以用其他插件来控制权限。

有名的权限控制插件有很多，例如：

GroupManager插件：
给权限组添加权限：/mangaddp 权限组(默认default) 权限
删除权限组权限：/mangdelp 权限组(默认default) 权限 或 /mangaddp 权限组(默认为default) -权限(注意权限前面的减号)
给单独玩家添加权限：/manuaddp 玩家ID 权限
删除单独玩家权限：/manudelp 玩家ID 权限 或 /manuaddp 权玩家ID -权限(注意权限前面的减号)

LuckPerms插件：
给权限组添加权限：/lp group 权限组(默认default) permission set 权限 true
删除权限组权限：/lp group 权限组(默认default) permission set 权限 false
给单独玩家添加权限：/lp user 玩家ID permission set 权限 true
删除单独玩家权限：/lp user 玩家ID permission set 权限 false

(参考资料：https://www.bilibili.com/read/cv9455406)

其中，GroupManager是Essentials插件中的一个分支，是许多人常用的一个权限组管理。现在似乎已是一个独立的插件了？
